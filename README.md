# Mapa da Violência e Desenvolvimento Econômico

Este repositório contém os dados e o código-fonte desenvolvidos para a Pesquisa Curricularizada de Graduação (PCG) do 3º semestre para a Universidade Católica de Santos. 
O projeto realiza uma análise exploratória para investigar a correlação entre o Produto Interno Bruto (PIB) e os índices de criminalidade nos 100 municípios mais ricos do Brasil (2015-2020).

## Tecnologias Utilizadas
* **Python 3**
* **Pandas** (Tratamento e manipulação de dados)
* **Plotly, Matplotlib e Seaborn** (Visualização e geração de gráficos)
